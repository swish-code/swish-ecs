# 30-Day Pilot MVP Plan

## Week 1
- Scaffold app shell
- Add auth and RBAC foundation
- Add Snowflake connection layer
- Create admin pages for brands, departments, users, and roles
- Deploy MVP schema and views

## Week 2
- Build SOP register
- Add file upload metadata capture
- Add SOP approval actions
- Add brand + department assignment workflow
- Build implementation tracker

## Week 3
- Build checklist template setup
- Build audit execution flow
- Build corrective action tracker
- Add evidence attachment flow

## Week 4
- Build KPI setup and monthly results
- Wire dashboard views
- Add audit trail writes
- Add Outlook notification hooks
- Test end-to-end workflow and demo readiness

## Acceptance target
The pilot must demonstrate:
- SOP creation and approval
- brand + department assignment
- implementation tracking
- food safety audit execution
- corrective action tracking
- evidence attachment
- KPI reporting
- dashboard visibility
